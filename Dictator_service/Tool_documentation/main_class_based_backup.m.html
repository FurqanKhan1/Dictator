main_class_based_backup module

@Author :Furqan Khan @Email :furqankhan08@gmail.com @Date :12/30/2016

Objective : The purpose of this file /module /Class is to actually execute the nmap process as a sub process (python module) and then colect the findings from the discovery output and save the details in the database table. This module implements start ,stop and resume functionality of nmap scanning

This class is used with both CLI and GUI mode

self)

Objective : This method is the polling method and acts like a thread schedular initializer.
